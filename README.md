# ECM1417 Space Exploration Website

Homepage link :   https://angelgebretensai.github.io/wedev3/
 

## Overview
This project is a static website for the ECM1417 module, containing four pages:
- `index.html` (Home)
- `history.html`
- `future.html`
- `contact.html`

All pages share a consistent navigation bar, footer, and styling via `style.css`.

## Deployment
The site is publicly hosted using **GitHub Pages** and also deployed inside the Azure Lab VM using:

python3 -m http.server 8000
